## Star Wars Unlimited TCG Card Downloader

This project contains a non finished cpp, and a finished python app with GUI
that lets you download high definition images of cards, which you can later use
to make your own proxies and print your own cards.

---

### Features

- GUI for easier use
- Windows and Linux support
- Super light weight
- Highest quality images (at least what i could find)
- Multiple art variants for the same card
- Card back-side support (downloads images' back-side if there's one)
- Choose which sets to download
- Download promo cards, which are unavailable otherwise

----

### Install

**Windows:**

- From realeses download the `.exe` file
- Probably have to use *"Run as Administrator"* to run the app (I was too
  lazy set up the apps authentication and metadata, so Windows stops you from
  running it by default)

**Linux:**
- From realses download the app file
- Might have to give permission to the file to run it: `chmod +x`

----

### Running

- Just double click the execeutable

---

### Creating a proxy (PDF to print)

1. Download the cards you want to use
2. Create a proxy file (I usually use [proxxied](proxxied.com))
3. Download the PDF file (make sure to put DPI / quality to max)
4. Print it on a thick paper with high print quality set in your printer driver
5. Laminate your printed pages
6. Cut pagaes into cards
7. Sleeve them
8. Enjoy :)
